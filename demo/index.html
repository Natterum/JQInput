JQInput


Скачать v0.1 на GitHub


JQInput является инструментом, который предлагает хороший и легкий способ управлением числами в inputbox. Плагин построен на JasaScript с использованием фреймворка JQuery.


Основные возможности:

  • Установка минимального и максимального диапазона для числа.
  • Разделение числа на разряды.

Примеры:

Вы можете установить минимальное и максимальное значение числа:

<script type="text/javascript">
	$(document).ready(function() {
	    	$('#input_1').JQInput({
	    		'min' 	: 0,
	    		'max' 	: 10000000000,
	    	});
	});
</script>

С помощью опции "spaces" можно разделить число на разряды. При этой опции курсор реагирует адекватно:

<script type="text/javascript">
	$(document).ready(function() {
	    	$('#input_2').JQInput({
	    		'min' 	: 0,
	    		'max' 	: 10000000000,
	    		'spaces': true,
	    	});
	});
</script>
